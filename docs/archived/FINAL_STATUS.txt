HOW TO USE
================================================================================

1. TRANSFORM CONTENT:
   POST /transform
   {
     "content": "def hello(): return 'world'",
     "source_domain": "code",
     "target_domain": "text"
   }

2. SEARCH WITH TRIAD-AWARE RANKING:
   POST /unified/search
   {
     "query": "How to deploy an API?",
     "k": 5,
     "triad_target": "concrete"
   }

3. GENERATE ANSWER WITH GEMINI:
   POST /unified/answer
   {
     "query": "What is artificial intelligence?",
     "k": 5,
     "triad_target_mode": "abstract"
   }

4. ENCODE MULTIMODAL CONTENT:
   POST /unified/multimodal
   {
     "texts": ["Text content"],
     "codes": ["def foo(): pass"],
     "return_triads": true
   }
