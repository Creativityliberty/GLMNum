================================================================================
TROUBLESHOOTING
================================================================================

‚ùå "API is not online"
   ‚Üí Make sure backend is running: python backend.py
   ‚Üí Check port 8081: lsof -i :8081
   ‚Üí Verify: curl http://localhost:8081/health

‚ùå "Cannot connect to localhost:8000"
   ‚Üí Make sure web UI server is running: python -m http.server 8000
   ‚Üí Check port 8000: lsof -i :8000

‚ùå "Port already in use"
   ‚Üí Kill all processes: python cleanup.py
   ‚Üí Then restart services

‚ùå "API returns 500 error"
   ‚Üí Check backend console for error messages
   ‚Üí Verify all dependencies are installed

================================================================================
CLEANUP & RESTART
================================================================================

To stop all services and reset:
  python cleanup.py

This will:
  ‚úÖ Kill all Python processes
  ‚úÖ Free all ports (8000-8090)
  ‚úÖ Reset the system

Then restart:
  1. Start backend: python backend.py
  2. Start web UI: python -m http.server 8000
  3. Open browser: http://localhost:8000
