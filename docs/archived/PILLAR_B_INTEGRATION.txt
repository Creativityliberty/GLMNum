================================================================================
T_VIS VECTOR COMPONENTS
================================================================================

T_vis = [d_emb, d_triad, d_scale, d_position]

1. d_emb (Embedding Distance)
   - Cosine distance between visual embeddings
   - Captures semantic similarity

2. d_triad (Triad Distance)
   - L1 distance on ∆∞Θ
   - Captures abstraction level difference

3. d_scale (Scale Distance)
   - Log ratio of image scales
   - Captures size transformation

4. d_position (Position Distance)
   - Euclidean distance between centers
   - Captures spatial transformation
