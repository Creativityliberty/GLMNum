USAGE
================================================================================

1. START BACKEND:
   python backend.py
   or
   uvicorn backend:app --host 0.0.0.0 --port 8081

2. ACCESS DOCUMENTATION:
   http://localhost:8081/docs (Swagger UI)
   http://localhost:8081/redoc (ReDoc)

3. TEST ENDPOINTS:
   # Health check
   curl http://localhost:8081/health

   # Transform
   curl -X POST http://localhost:8081/transform \
     -H "Content-Type: application/json" \
     -d '{"content":"def hello(): pass","source_domain":"code","target_domain":"text"}'

   # Search
   curl -X POST http://localhost:8081/unified/search \
     -H "Content-Type: application/json" \
     -d '{"query":"How to deploy?","k":5,"triad_target":"concrete"}'

   # DeepTriad
   curl -X POST http://localhost:8081/deeptriad/analyze \
     -H "Content-Type: application/json" \
     -d '{"chunks":["First chunk","Second chunk"]}'
